## Научиться учиться

Небольшой учебный одностраничный сайт о современных и эффективных подходах к обучению. Проект выполнен в рамках обучения верстке: отработка семантической разметки, адаптивной вёрстки и методологии БЭМ.

### Стек и особенности

- **HTML5**: семантическая разметка, блоки `header`, `main`, `section`, `footer`.
- **CSS3**: flex‑разметка, позиционирование, базовая анимация.
- **Методология БЭМ**: файловая структура по блокам в папке `blocks`, модификаторы, элементы.
- **Чистый фронтенд**: без JavaScript и сборщиков, только статика.

### Структура проекта

- **`index.html`** – основная страница сайта.
- **`pages/index.css`** – главный файл стилей, который подключает стили блоков.
- **`blocks/`** – стили, разбитые по БЭМ‑блокам (например, `header`, `footer`, `cards`, `two-columns` и т.д.).
- **`images/`** – изображения и иконки, используемые на странице.
- **`vendor/normalize.css`** – сброс и выравнивание стилей в разных браузерах.

### Как запустить проект локально

1. **Клонируйте репозиторий** (или скачайте архив проекта):
   ```bash
   git clone <url-репозитория>
   cd how-to-learn-main
   ```
2. **Откройте файл `index.html` в браузере**:
   - двойным кликом по `index.html`, или
   - через контекстное меню «Открыть с помощью…» → ваш браузер.

Дополнительный вариант — использовать любой простой локальный сервер (например, расширение Live Server в VS Code), но это не обязательно.

### Деплой

Так как это статический сайт, его можно опубликовать на:

- **GitHub Pages**
- **Netlify**
- **Vercel**

Достаточно выложить содержимое папки проекта как статический хостинг и указать корнем `index.html`.

### Планы по доработке (опционально)

- **Адаптивная вёрстка** под разные устройства (мобильные, планшеты, десктоп).
- **Доступность**: улучшение ARIA‑атрибутов, контрастности и навигации с клавиатуры.
- **Мелкие улучшения UI/UX**: плавные анимации, состояния наведения, фокус и т.п.
